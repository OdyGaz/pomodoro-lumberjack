# 🪓 Pomodoro Lumberjack

A simple, interactive Pomodoro timer web application designed to help you stay focused.
Instead of a standard progress bar, it features a lumberjack who chops wood during your work sessions and rests during your breaks.

## 🚀 Live Demo
👉 **You can try the live application here: [Online App](https://odygaz.github.io/pomodoro-lumberjack/)**

🤖 **Download the Android App (Offline APK): [Download APK](https://github.com/OdyGaz/pomodoro-lumberjack/releases/download/v1.0.0/pomodoro-lumberjack.apk)**

## ✨ Features
* **Customizable Intervals:** Set your preferred work and break times easily.
* **Interactive Visuals:** 
  * 🪓 **Work Session:** The lumberjack is active and chopping wood (animated GIF).
  * ☕ **Break / Pause / Reset:** The lumberjack stops and rests (static PNG).
* **Cycle Counter:** Keeps track of your completed work sessions.
* **Responsive Design:** Clean and minimalist interface that works on both desktop and mobile devices.

## 🛠️ Technologies Used
* **HTML5:** Semantic structure of the application.
* **CSS3:** Responsive card layout, custom styling, and transition effects.
* **JavaScript (ES6):** Timer countdown logic, state management, and dynamic image switching.
